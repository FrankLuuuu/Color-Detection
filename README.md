# Install packages (if missing):
```
pip(3) install pandas
pip(3) install opencv-python
```

# Run color detection model:
```
python(3) color_detection.py -i [IMAGE]
```
### For example, try running:
```
python3 color_detection.py -i colorpic.jpg
```
